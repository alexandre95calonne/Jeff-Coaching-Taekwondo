# 🥋 Jeff Coaching Taekwondo

Showcase website built for a professional Taekwondo coach.  
Goal: modern image, smooth UX, and instant booking via **Calendly** integration.

📱 Fully responsive, clean interface, fast and intuitive.  
🛠️ Tech stack: **Next.js**, **Tailwind CSS**, **Supabase**, **Next UI**.
